# MinPad v1.1.0 - 快速开始指南

## 🚀 5 分钟快速开始

### 第一步：运行应用
```bash
java -jar minpad-1.0.0.jar
```
你会看到系统托盘出现一个蓝色的 "#" 图标。

### 第二步：配置快捷键
1. **右键点击** 托盘图标
2. 选择 **"设置快捷键..."**
3. 点击你要配置的按键旁的 **"编辑"** 按钮

### 第三步：填写配置
在弹出的对话框中填写（至少填一项）：
- **名称**：操作的描述（例如："打开记事本"）
- **命令**：程序路径（例如："notepad.exe"）
- **组合键**：按键组合（例如："ctrl+s"）

### 第四步：保存并使用
1. 点击 **"确定"** 按钮
2. 配置会立即保存
3. 使用数字键盘按键触发快捷操作

## ⚙️ 配置文件位置

你的所有配置会自动保存到：
```
C:\Users\[你的用户名]\.minpad\config.json
```

### 配置持久化说明
- ✅ **自动保存** - 修改配置后立即保存，无需手动保存
- ✅ **自动加载** - 重启应用后自动恢复之前的配置
- ✅ **JSON 格式** - 配置文件是标准 JSON，可以手动编辑（不推荐）

## 🔄 常见操作

### 重置某个按键为默认
1. 打开设置对话框
2. 找到要重置的按键
3. 点击 **"重置"** 按钮（仅对 +/-/Enter 可用）

### 完全重置所有配置
```bash
# 方法 1: 删除配置文件，重启应用
删除: C:\Users\[用户名]\.minpad\config.json

# 方法 2: 手动删除文件夹
删除: C:\Users\[用户名]\.minpad\
```

### 备份配置
```bash
# 复制配置文件到安全位置
copy "C:\Users\[用户名]\.minpad\config.json" "D:\backup\config.json"
```

### 恢复备份
```bash
# 从备份恢复
copy "D:\backup\config.json" "C:\Users\[用户名]\.minpad\config.json"
```

## 🆘 故障排除

### 问题：按键无响应
**解决方案：**
1. 确保 NumLock 键已开启（看键盘上的指示灯）
2. 尝试重启应用
3. 检查应用是否在系统托盘中

### 问题：命令无法执行
**解决方案：**
1. 确认程序路径正确
2. 使用完整路径，如 `C:\Program Files\...\app.exe`
3. 对于组合键，确保格式正确（如 `ctrl+s`）

### 问题：配置无法保存
**解决方案：**
1. 检查 C:\Users\[用户名]\.minpad\ 目录权限
2. 确保磁盘有足够空间
3. 使用管理员身份运行应用

### 问题：配置文件损坏
**解决方案：**
1. 删除 `C:\Users\[用户名]\.minpad\config.json`
2. 重启应用（会使用默认配置）
3. 重新配置快捷键
